# Projet-Site-Caritative
Nous avons crée un site fictif qui avait pour but caritatif pour un Projet de Premier semestre en première année de But informatique

## Description
Ce projet académique consistait à faire un site caritative basée sur une demande d'un client fictif (autres groupe dans notre promo), le client avait demandé de faire un site avec la promotions d'un événement associé à un cadre environnemental il s'appelle "Bataille de semance" le but consistait de faire une bataille avec des boules d'eau remplis d'engrais (semances) en pleine forêt.

Style du site:

- Rouge et blanc reprise des couleurs du site de la croix rouge

- boutons verts pour rappeler l'environnement

## Fonctionnalités
-- page d'accueil--
Il y'a une page d'accueil contenant des descriptifs de l'événement (dates, lieux, objectifs) deux boutons
- 1er pour faire des dons
- 2ème pour voir le concept

-- Concept --
Descriptif de l'événement

-- Contact --
3 cases
- 1er mettre son nom
- 2ème Email
- 3ème écrire un message

puis un bouton pour envoyé un message avec tout les infos

-- Dons --
Dans la page de dont nous avons une case pour mettre le montant de la donnation et un bouton pour faire le dons.
## Technologies
Uniquement du front-end

HTML5/CSS3/javascript

## Utilisation

- Découvrir l'événement : Consulter les objectifs de la "Bataille de semence" (dates, lieux et concept écologique) sur la page d'accueil et la page Concept.

- Soutenir la cause : Accéder au formulaire de don pour simuler une contribution financière.

- Nous contacter : Utiliser le formulaire dédié pour envoyer un message aux organisateurs (nom, email et message requis).

- Navigation fluide : Utiliser le menu de navigation pour circuler entre les différentes pages HTML sans recharger manuellement le navigateur.


## Configuration
Comment lancer le site

utiliser Quick PHP mettez les parmètres suivants:

- Binding adresse : 127.0.0.1
- Port : 80
- Selectionner le dosier du projet

Après ça vous tapez ceci 127.0.0.1/HTML/index.html dans la barre de recherche Google
